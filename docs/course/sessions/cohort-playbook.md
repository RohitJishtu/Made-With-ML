# Cohort Playbook

## Lifecycle

Plan → Setup → Run (10 weeks) → Demo days → Retro → Alumni sessions

## Setup checklist (T-1 week)

- [ ] Setup clinic scheduled
- [ ] Week 1 guide posted
- [ ] For Week 9: confirm learners have Ollama or Docker

## Weekly checklist

- [ ] Post week link Monday
- [ ] Live session Wednesday
- [ ] Office hours Thursday
- [ ] Deliverable reminder Friday

## Mid-cohort pulse (Week 4)

- Survey: pace, difficulty, blockers
- Identify at-risk learners (no repo, no MLflow runs)

## After cohort

- [ ] Retrospective within 1 week
- [ ] Update [CHANGELOG](../CHANGELOG.md)
- [ ] First alumni session within 4 weeks
