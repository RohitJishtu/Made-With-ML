## Program formats

### Format A: Core MLOps (8 weeks)
Weeks 1–8 only. Best for teams new to production ML.

### Format B: MLOps + AIOps (10 weeks)
Full program. Week 8 MLOps capstone + Weeks 9–10 AIOps extension.

### Format C: Split cohorts
- **Cohort 1** (Sessions 1–8): MLOps
- **Break** (2–4 weeks): alumni office hours
- **Cohort 2** (Sessions 9–10): AIOps — alumni from Cohort 1 welcome back

### Format D: Monthly alumni sessions
After any cohort graduates, run monthly 2h sessions from [alumni-track.md](alumni-track.md).

## Suggested live session cadence

| Event | Frequency | Duration |
|-------|-----------|----------|
| Kickoff | Once | 60 min |
| Weekly live lab | Weekly | 90 min |
| Office hours | 2× per week | 45 min |
| Demo day | Week 8 and Week 10 | 2 h |
| Alumni session | Monthly | 90 min |

## Register your cohort

Copy the template in [cohort-registry.md](cohort-registry.md) and add a row for each run. This helps future facilitators learn from past sessions.

## Contributing session improvements

After each cohort:

1. File feedback via GitHub Issues (label: `cohort-feedback`)
2. Update [CHANGELOG.md](../CHANGELOG.md) with what changed
3. Propose roadmap items in [roadmap.md](roadmap.md)
