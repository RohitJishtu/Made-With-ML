# Facilitator Guide

## Before cohort

- [ ] Pick 8-week or 10-week format
- [ ] [Register cohort](cohort-registry.md)
- [ ] Create Slack/Discord channel
- [ ] Run a 45-min setup clinic (Python venv, Git, GPU options)

## Weekly rhythm

| Day | Action |
|-----|--------|
| Mon | Post week guide link |
| Wed | 90-min live session ([templates](session-templates.md)) |
| Thu | Office hours |
| Fri | Deliverable reminder |

## Live session structure (90 min)

```
0:00 – 0:10   Recap
0:10 – 0:25   Concepts
0:25 – 0:65   Guided lab (learners work in their own repos)
0:65 – 0:80   Q&A
0:80 – 0:90   Homework preview
```

## Demo days

- **Week 8** — MLOps capstone (5–7 min per learner)
- **Week 10** — AIOps capstone (eval scores + guardrails required)

## After cohort

- [ ] Update [CHANGELOG](../CHANGELOG.md)
- [ ] Log outcomes in [cohort-registry.md](cohort-registry.md)
- [ ] Schedule first [alumni session](alumni-track.md)
