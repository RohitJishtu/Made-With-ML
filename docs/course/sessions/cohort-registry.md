# Cohort Registry

Log every cohort run here so future facilitators can learn from past sessions. Add a new row at the top.

## How to register

Copy this template when you announce a cohort:

```markdown
### Cohort: [Name] — [Season Year]

| Field | Value |
|-------|-------|
| Facilitator(s) | |
| Format | 8-week MLOps / 10-week MLOps+AIOps |
| Start date | YYYY-MM-DD |
| End date | YYYY-MM-DD |
| Learners (started / finished) | / |
| Channel | Discord / Slack / GitHub Discussions link |
| Demo Day recordings | link or N/A |
| Top blockers | |
| Changes for next cohort | |
| Alumni sessions planned | yes / no |
```
