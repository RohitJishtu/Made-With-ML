# Exercise Bank

Extra practice if you finish a week early.

## MLOps

| Week | Exercise |
|------|----------|
| 1 | Draw failure modes for your system (data outage, model crash, latency spike) |
| 2 | Create a deliberately bad dataset; confirm your validation suite catches it |
| 3 | Train with frozen vs unfrozen encoder; compare MLflow curves |
| 4 | Plot a confusion matrix for holdout predictions |
| 5 | Add `GET /health` returning model version and uptime |
| 6 | Fail CI if test coverage drops below 80% |
| 7 | Multi-stage Dockerfile to reduce image size |
| 8 | Write a Prometheus alert rule for p95 latency > 500ms |

## AIOps

| Week | Exercise |
|------|----------|
| 9 | Compare RAG vs zero-shot on 5 questions; log when RAG helps |
| 10 | Add promptfoo to CI — fail PR if faithfulness drops |
| 10 | Build a simple agent that routes between classifier and RAG |
