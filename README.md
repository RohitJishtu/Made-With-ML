# AI_ML_Ops

Open source **MLOps + AIOps** curriculum — 10 weeks, no vendor lock-in.

Learn to build, deploy, and operate production ML and LLM systems using Ray, MLflow, Docker, Kubernetes, Ollama, and more.

## Quick start

```bash
git clone https://github.com/RohitJishtu/AI_ML_Ops.git
cd AI_ML_Ops
```

1. Read [docs/course/index.md](docs/course/index.md)
2. Begin [Week 1](docs/course/week-01-foundations.md)
3. Use [data/samples/](data/samples/) for practice exercises

**Time:** ~10–15 hours per week.
