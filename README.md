## How to use

This repository is **curriculum only**. Each week links to official open source documentation. Learners build projects in their own repos.

```bash
git clone https://github.com/YOUR_USERNAME/AI_ML_Ops.git
cd AI_ML_Ops
# Read docs/course/week-01-foundations.md and begin
```

**Time:** ~10–15 hours per week.
